# Timezone Information Retrieval

This repository contains a simple web application that allows users to retrieve timezone information using geographical coordinates and addresses. It utilizes the [Geoapify API](https://www.geoapify.com/) to fetch timezone data.

## Features

- Display user's current timezone based on their geographical coordinates.
- Retrieve timezone information by entering an address.
- User-friendly interface with input fields and buttons.
- Utilizes HTML5 Geolocation API for automatic retrieval of user coordinates.
- Responsive design for various screen sizes.
